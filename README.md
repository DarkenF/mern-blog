# mern-blog

Для корректной работы приложения на компьютере должен быть установлен node.js 
Данее необходимо установить yarn для более комфортной работы с консолью. 
0. Команда: npm i -g yarn 

Перед запуском приложения необходимо установить все зависимости:
1. yarn install 
Далее переходим в папку клиента: 
2. cd ./client
Устанавливаем зависимости для клиента:
3. yarn install
Переходим на уровень вверх и запускаем проект:
4. cd ../
5. yarn dev
